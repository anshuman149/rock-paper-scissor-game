# Rock-Paper-Scissors Game 🎮

A fun and interactive Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Features
- 🎨 Clean, responsive user interface
- 🖱️ Interactive gameplay with instant feedback
- 📊 Score tracking and winner announcements
- 🔊 Optional sound effects for added excitement

## How to Play
1. Open `roclpappergame.html` in your browser.
2. Choose Rock, Paper, or Scissors.
3. The computer makes its move.
4. The winner is announced, and scores are updated automatically.
